
# IPL Match Intelligence Dashboard

## Project Overview

The IPL Match Intelligence Dashboard is an interactive Power BI project developed to analyze Indian Premier League (IPL) data and uncover meaningful insights related to teams, players, match outcomes, venue performance, and advanced player analytics.

The dashboard transforms raw ball-by-ball and match-level data into actionable insights using Power BI, DAX, Power Query, and data modeling techniques.

---

## Objectives

* Analyze team and player performance across IPL seasons.
* Identify top-performing batters and bowlers.
* Evaluate toss impact and match-winning strategies.
* Analyze venue-based scoring patterns.
* Discover elite finishers and death-over specialists.
* Understand batting styles and power-hitting capabilities.

## Advanced Metrics Implemented

### Finisher Score

Used to identify impactful finishers based on death-over performance.

Formula:

Finisher Score = (Death Over Strike Rate × 70%) + (Runs Scored × 30%)

### Death Bowler Score

Used to identify effective death-over bowlers.

Formula:

Death Bowler Score = (Wickets × 30%) - (Runs Conceded × 70%)

### Dot Ball Percentage

Used to identify bowlers who consistently create pressure.

Formula:

Dot Ball Percentage = (Dot Balls / Total Legal Deliveries) × 100

### Batting Style Analysis

Runs were classified into:

* Four Runs
* Six Runs
* Running Runs (1s, 2s, and 3s)

This helps identify batting patterns and scoring behavior.
